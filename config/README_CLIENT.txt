Это тестовый pack, чтобы проверить сам лаунчер.

Как сделать реальный клиент:
1. Замени start-client.bat на свой BAT запуска NeoForge.
2. Добавь mods, config, resourcepacks и остальные файлы в pack.
3. Сгенерируй новый client-manifest.json через tools/New-VeltrixManifest.ps1.
4. Раздай pack по HTTP или положи локально рядом с launcher.
